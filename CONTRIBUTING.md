# Contributing to Lollypop Shop

Thank you for considering a contribution! Here is how to get started quickly.

## Quick start

- Fork the repo and create a feature branch.
- Install deps: `npm install`.
- Run dev server: `npm run dev`.
- Run tests before pushing: `npm test`.
- Build check (optional but recommended): `npm run build`.

## Commit and PR guidelines

- Keep changes focused; split unrelated work into separate PRs.
- Add or update tests when fixing bugs or adding features.
- Describe the change and testing done in the PR body.

## Environment

- Copy `.env.example` to `.env` and fill in values needed for your run (Firebase optional; email vars for Netlify function).

## Reporting issues

- Include steps to reproduce, expected vs. actual behavior, and environment (OS, Node version).
- If relevant, attach console/network logs or screenshots.
